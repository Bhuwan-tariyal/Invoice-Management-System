About Invoice Manager

A simple and powerful invoice generation system built for small businesses and freelancers.

📄 What It Does

This application helps you create, manage, and download professional invoices in PDF format with just a few clicks.

⚡ Key Features

  • Create invoices instantly
  • Auto calculation of amounts
  • Download as PDF
  • Customer-based invoice tracking

🎯 Purpose

Built to simplify billing workflow and reduce manual effort for businesses handling multiple clients.

👨‍💻 Tech Stack

Spring Boot, Java, HTML, CSS, JavaScript, REST APIs, PDF generation

This system is designed with scalability in mind and can be extended to include payment tracking, GST reports, and analytics dashboard in future updates.
